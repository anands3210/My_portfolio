## ⚙️ Setup and Usage

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/your-repo-name.git
    ```
    *(Replace `your-username/your-repo-name` with your actual GitHub repo details)*
2.  **Navigate to the project directory:**
    ```bash
    cd your-repo-name
    ```
3.  **Open `index.html`:** Simply open the `index.html` file in your preferred web browser.

No complex build steps or dependencies are required.

## ✏️ Customization

If you want to adapt this portfolio for your own use:

1.  **Edit `index.html`:**
    *   Update all text content (name, bio, descriptions, education details, project details, contact links).
    *   Replace placeholder links (`href`) with your actual links.
    *   Replace image `src` attributes if using different image names/paths.
    *   Update the `download` attribute and `href` for the resume link.
2.  **Edit `style.css`:**
    *   Modify colors, fonts, spacing, or layout as desired. Key color variables are defined in the `:root` section.
3.  **Replace Assets:**
    *   Add your own profile picture (`dp1.png`).
    *   Add your own project screenshots (`project1.png`, `project2.png`).
    *   Add your own resume file (`anand_resume.pdf`).


Feel free to reach out if you have any questions or feedback!
